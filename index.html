// ═══════════════════════════════════════════════════════════ // BACKEND - PEDIDOS DE ATENCIÓN PRODUCCIÓN A MANTENIMIENTO - AKAPOL // ═══════════════════════════════════════════════════════════ // Esta planilla necesita UNA sola pestaña llamada exactamente: Pedidos // El texto de los encabezados de la fila 1 NO importa (no se lee por nombre) — // lo que importa es el ORDEN de las columnas, que tiene que ser exactamente: // A=id | B=fecha | C=hora | D=turno | E=quienLlamo | F=equipo | G=problema | // H=urgencia | I=estado | J=notaResolucion | K=fechaAtencion // Posiciones fijas de columna (empezando en 0 = columna A) var COL = { id: 0, fecha: 1, hora: 2, turno: 3, quienLlamo: 4, equipo: 5, problema: 6, urgencia: 7, estado: 8, notaResolucion: 9, fechaAtencion: 10 }; function doGet(e) { var ss = SpreadsheetApp.getActiveSpreadsheet(); var pedidos = leerHoja(ss.getSheetByName('Pedidos')); return ContentService.createTextOutput(JSON.stringify({ pedidos: pedidos })) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { try { var body = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); var hoja = ss.getSheetByName('Pedidos'); if (body.action === 'nuevo') { hoja.appendRow([ body.id || Utilities.getUuid(), body.fecha, body.hora, body.turno, body.quienLlamo, body.equipo, body.problema, body.urgencia, 'Pendiente', '', '', ]); return respOk({ ok: true }); } if (body.action === 'atender') { var data = hoja.getDataRange().getValues(); var encontrado = false; for (var i = 1; i < data.length; i++) { if (String(data[i][COL.id]) === String(body.id)) { hoja.getRange(i + 1, COL.estado + 1).setValue('Finalizado'); hoja.getRange(i + 1, COL.notaResolucion + 1).setValue(body.nota || ''); hoja.getRange(i + 1, COL.fechaAtencion + 1).setValue(body.fechaAtencion || ''); encontrado = true; break; } } return respOk({ ok: true, encontrado: encontrado }); } return respOk({ ok: false, error: 'acción no reconocida' }); } catch (err) { return respOk({ ok: false, error: err.toString() }); } } function respOk(obj) { return ContentService.createTextOutput(JSON.stringify(obj)) .setMimeType(ContentService.MimeType.JSON); } function valorCelda(val, campo) { if (val instanceof Date) { if (campo === 'fecha') return Utilities.formatDate(val, Session.getScriptTimeZone(), 'dd/MM/yyyy'); if (campo === 'hora') return Utilities.formatDate(val, Session.getScriptTimeZone(), 'HH:mm'); return Utilities.formatDate(val, Session.getScriptTimeZone(), 'dd/MM/yyyy HH:mm'); } return val; } function leerHoja(sheet) { var data = sheet.getDataRange().getValues(); data.shift(); // saca la fila de encabezados, sea cual sea el texto que tengan return data .filter(function (row) { return row[COL.id] !== '' && row[COL.id] !== null; }) .map(function (row) { return { id: valorCelda(row[COL.id], 'id'), fecha: valorCelda(row[COL.fecha], 'fecha'), hora: valorCelda(row[COL.hora], 'hora'), turno: valorCelda(row[COL.turno], 'turno'), quienLlamo: valorCelda(row[COL.quienLlamo], 'quienLlamo'), equipo: valorCelda(row[COL.equipo], 'equipo'), problema: valorCelda(row[COL.problema], 'problema'), urgencia: valorCelda(row[COL.urgencia], 'urgencia'), estado: valorCelda(row[COL.estado], 'estado'), notaResolucion: valorCelda(row[COL.notaResolucion], 'notaResolucion'), fechaAtencion: valorCelda(row[COL.fechaAtencion], 'fechaAtencion'), }; }); }
